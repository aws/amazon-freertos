FreeRTOS API Reference

  FreeRTOS is a secure, cloud-native IoT operating system for microcontrollers. The library-specific API references for the libraries below are generated through the AWS Reference Integrations Github.

  • Bluetooth Low Energy

  • Platform

  • POSIX

  • Wi-Fi

  • HTTPS

  • AWS IoT OTA Agent

  • Secure Sockets

  • PKCS #11 API

    Each API reference outlines:

    • The functions calls

    • The parameters used in the function calls

    • The parameter types used

    Please use these APIs to help implement your application functionality. The APIs have abstracted out many hardware dependencies thereby allowing them to be used on various microcontroller platforms that are supported by FreeRTOS.
