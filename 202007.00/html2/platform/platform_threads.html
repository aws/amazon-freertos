Thread Management

Threading and synchronization functions used by libraries in this SDK.

The platform thread management component provides other libraries with functions relating to threading and synchronization. It interfaces directly with the operating system to provide:

  • A function to create new threads.
  • Synchronization mechanisms such as mutexes and counting semaphores.

Dependencies

Dependencies of the platform thread management component.

Thread management direct dependencies

Currently, the platform thread management component has the following dependencies:

  • The operating system must provide the necessary APIs to implement the thread management functions.
  • The logging library may be used if IOT_LOG_LEVEL_PLATFORM is not #IOT_LOG_NONE.
