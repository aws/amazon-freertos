AWS IoT Device SDK for Embedded C
AWS IoT Device SDK for Embedded C API Reference

  The AWS IoT Device SDK for Embedded C is a collection of C source files that can be used in embedded applications to securely connect to the AWS IoT platform.

  The following AWS IoT Device SDK for Embedded C libraries are documented in library-specific API references. Each library shares APIs with an FreeRTOS library of the same name.

  • Atomic Operations

  • Linear Containers

  • Logging

  • MQTT

  • AWS IoT Device Shadow

  • Static Memory

  • Task Pool

  Each API reference outlines:

  • The functions calls

  • The parameters used in the function calls

  • The parameter types used
