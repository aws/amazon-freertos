◆ AWS_IOT_PROVISIONING_FLAG_ENCRYPT_CREDENTIALS

#define AWS_IOT_PROVISIONING_FLAG_ENCRYPT_CREDENTIALS   0x00000001

Flags that modify the behavior of Provisioning library functions.

Provisioning Function Flags

Flags should be bitwise-ORed with each other to change the behavior of Provisioning library functions.

The following flag is valid for the Provisioning CreateKeysAndCertificate function: AwsIotProvisioning_RegisterThing

  • AWS_IOT_PROVISIONING_FLAG_ENCRYPT_CREDENTIALS
    Flags that modify the behavior of Provisioning library functions. Flag for requesting encryption of device credentials in server response.
Note
Set the flags parameter passed to AwsIotProvisioning_RegisterThing to this value when using to enable credentials encryption in the response from the server.
